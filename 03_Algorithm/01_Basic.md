# 알고리즘의 표현

- 의사코드 (슈더코드)
  - 코드가 돌아가는 문법과 비슷한 형태로 작성
  - 실제로 작동하지는 않지만 구현되기 전에 알고리즘의 형태를 보여줌

```python
def CalcSum( n ) :
    sum <- 0;
    for i in range (1, n+1) :
        sum <- sum + i;
    return sum;
```

- 순서도
  - 알고리즘을 도형의 요소로 표현



# 알고리즘의 성능

- 정확성
- 작업량
  - 시간 복잡도
    - 실제 걸리는 시간을 측정
    - 실행되는 명령문을 계산
    - 공간에 대한 비용이 컸던 과거에 비해 시간복잡도가 더욱 중요해짐
    - 빅-오 표기법
      - O(n의 최고차항)
      - 최악의 경우를 가정

- 메모리 사용량
  - 공간 복잡도

- 단순성
- 최적성
