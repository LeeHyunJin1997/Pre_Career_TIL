# 크기 단위

- **픽셀 (px)**
  - 고정 크기
- **%**
  - 가변 크기
- **em**
  - 지정된 사이즈에 배수로 크기를 지정
  - 상속에 영향을 받음
- **rem**
  - 최상위 요소(html)의 사이즈를 기준으로 배수 단위를 가짐
- **viewport**
  - 유저에게 보이는 창의 영역
  - 디바이스를 기준으로 상대적인 사이즈가 결정됨
  - vw (창 너비 = 100 vw)
  - vh (창 높이 = 100 vh)
