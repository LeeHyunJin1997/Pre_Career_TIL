# 루트 계정과 IAM 계정

AWS는 루트 계정과 IAM 계정을 지원한다.

루트 계정은 AWS 서비스 및 리소스에 대한 모든 액세스 권한을 지닌 계정이다. 단, 루트 계정이 탈취당했을 때의 피해를 고려해 AWS는 공식적으로 루트 계정을 사용하지 않는 것을 권장한다.

그 대안으로 주어지는 것이 IAM 계정이다. IAM 계정은 하나의 AWS 계정을 공유하며, 각각 분리된 권한을 갖게 된다.
