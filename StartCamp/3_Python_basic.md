# 파이썬 챗봇

- 미세먼지봇

```python
dust = 70
#if문을 이용해, 미세먼지 수치에 맞는 대기 상태 출력

if dust > 150:
  print('매우나쁨')
elif 80 < dust <= 150:
  print('나쁨')
elif 30 < dust <= 80:
  print('보통')
else:
  print('좋음')
```
