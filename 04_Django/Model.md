# ORM

- Object-Relational-Mapping
- Django와 SQL 간의 데이터를 변환
- Django는 내장 Django ORM을 사용
- 현대 웹 프레임워크은 빠른 웹개발에 중점, 이런 목적에 적절
- 장점
  - SQL를 잘 알지 못해도 DB 조작 가능
- 단점
  - ORM만으로 완전한 SQL 기능을 구현하기 어려움
  - 효율성이 다소 떨어짐
