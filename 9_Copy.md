## 할당

- 대입연산자(`=`) 이용

- `x = y = [1,2,3]` 과 같이 대입 연산자를 통한 복사는 객체 참조도 복사
  - `x[0] = 3` 등으로 바꾸면 y도 함께 바뀜



# 얕은 복사 (Swallow Copy)

- `x = y[:]` 처럼 연산이나 처리를 거쳐 복사하면 형태만 가져올 수 있음
- 단, `[1, 2, [3]]` 처럼 이중 리스트를 슬라이싱하여 가져오면 [3]은 주소를 가져온 것.



# 깊은 복사 (Deep Copy)

```python
import copy
a = [1, 2, [3]]
b = copy.deepcopy(a)
```

- `copy.deepcopy()` 를 이용하면 모든 주소로 찾아 들어가 형태만 복사함
